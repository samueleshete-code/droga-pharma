#### \# 💊 Droga Pharma - Online Pharmacy System

#### 

#### A full-stack \*\*Online Pharmacy Management System\*\* built using \*\*PHP, MySQL, HTML, CSS, JavaScript, and AJAX\*\*.

#### 

#### This system allows users to browse medicines, add to cart, and manage orders efficiently.
